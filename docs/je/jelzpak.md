# Java版懒人包
懒人包包含了游玩所需的全部资源，方便快速上手。懒人包分为多个版本，您可根据您的电脑性能选择您想要的版本。

## 懒人包极低配版
该版本仅包含优化mod和必须mod，不包含画面增强mod，同时将游戏内视频设置调为极低，**适用于极低配置的电脑。**

:::danger 注意
如果您的电脑并不至于非常卡，则强烈不建议您使用此包。因为此包的画面表现极差无比，且缺少一部分功能类mod。建议您先尝试中配版，不流畅时再安装本包。（换包请先卸载旧包再安装新包）
:::
